## How to start server

<ol type="1">

<li> ssh Clone it from master branch </li>
<pre> git clone git@gitlab.com:paycron-dev/api.git </pre>
<li> Go to api repository </li>
<pre> cd ./api </pre>
<li> install dependencies </li>
<pre> pnpm i </pre>
<li> start project </li>
<pre> pnpm start </pre>
	
</ol>
